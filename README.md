# Description of base_os image

## Overview

Centos7の最新版をベースに下記のソフトウェアをインストールしています。

- gcc 7.4.0

Note that:  
the base_os:latest tag is always the most recent version currently available.

## How to use

下記のコマンドでダウンロードできます。

```bash
docker pull nagatax/base_os:latest
```
